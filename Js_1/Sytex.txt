- Основы JS - 
программы на JS называются скриптами. Они могут встраиваться в html и выполняться автоматически при загрузке 
веб страницы. Скрипты распространяются и выполняются как простой текст. Вот основное отличие JS от например 
python то что скриптам не нужна специальная подготовка или компиляция. 
Скрипт КАК подключить JS:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
    <script src="./JS.js"></script> --- в конце BODY!!!
</body>
</html>

- тема 1 -
- ПЕРЕМЕННЫЕ -
переменные - это именнованное хранилище данных.
существует 3 основных оператора объявления переменных
1) let - локальные переменные. 
2) var - глобальная переменныя.
3) const - константа (неизменяемая и не переопределяемая переменная).

пример реализации переменной и вывода её результата:
let message;
message = 'Hello, w0rld!';
alert (message);
сокращённый вариант:
let message = 'Hello, w0rld!';
alert (message);

чтобы не было проблем не юзай var, чаще всего let или const.

переменные объявленные с помощью var имеют глобальную область видимости в рамках всей структуры скрипта.
let и const имеют блочную область видмости и видны только в блоке кода в котором они объявлены:
{
 let
 const
}

- тема 2 -
- ТИПЫ ДАННЫХ -
переменные объявленные с помощью let или var могут быть изменены в дальнейшем. При этом стоит учитывать тип данных
переменной.
тип данных:
user = 'Deku'; - строка - string
age = 15; - числовой - number - int
IsAlive = true - булевый - bool
Weight = 62.7; - числовой - number - float
array = ['Mei', age, 3.14]; - массив - array
obj = {name: 'Iida', age: 5}; - объект
undefined - сообщение если используется переменная что была не объявлена.
null - ошибка пустоты, если в переменной ничего нет.
typeof - показывает какой тип данных переменной.
